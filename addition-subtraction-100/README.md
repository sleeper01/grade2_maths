# 100以内的加法和减法 - 二年级上册第二单元

> 配套人教版二年级数学上册 · 第二单元 · 家长辅导版 PPT 课件

## 🎯 适用场景

- **使用对象**：家长辅导二年级孩子学习
- **设计理念**：每节内容拆分成多张幻灯片，按"引入 → 演示 → 练习"节奏推进
- **风格**：可爱卡通（小熊老师）+ 大字号 + 高对比色，方便孩子集中注意力

## 📚 学习目标

完成后孩子应能：

1. 熟练计算**整十数加减整十数**（如 30 + 20、50 − 20）
2. 掌握**两位数加减一位数**（不进位/不退位）
3. 理解**进位加法**（个位满 10 向十位进 1）
4. 理解**退位减法**（个位不够减，从十位借 1 当 10）
5. 会用**竖式**计算两位数加减两位数
6. 能解决**比多比少**的实际问题

| 节 | 主题 | 张数 | 重点 |

| 0 | 封面 | 1 | 引入小熊老师 |

## 🚀 使用方法

1. 双击 `index.html` 用浏览器打开
2. 点击 **「开始学习 🚀」** 进入第 1 节
3. 在每个章节内，用屏幕下方的 **← →** 切换幻灯片
4. 每节最后一页时，按 **下一节 →** 进入下一章节
5. **键盘快捷键**：
   - `→` 下一页 / 下一节
   - `←` 上一页 / 上一节

## 💡 家长使用建议

- **每个 💡 家长提示** 都需要你大声念给孩子听
- **重播按钮（↻）**：孩子没看懂可点重播
- **互动练习（section 6）**：让孩子自己点选答案，做错不要直接告诉答案
- **每节最后一张** 都有速算小挑战，可以打印出来让孩子再做一遍
