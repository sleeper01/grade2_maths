# 二年级数学 第 8 单元：数学广角 - 搭配问题

## 单元概述
本单元通过生活中的搭配情境，让学生初步了解排列组合的基本思想，培养学生的有序思维和逻辑推理能力。

## 教学目标
1. 通过观察、操作等活动，找出简单事物的排列数和组合数
2. 学会用连线、列举等方法解决搭配问题
3. 培养有序、全面思考问题的意识
4. 感受数学与生活的密切联系

## 教学内容

### Section 1: 情境导入 - 衣服搭配
- 2 件上衣和 3 条裤子的搭配
- 用连线法展示所有搭配方案
- 理解乘法原理：2 × 3 = 6 种

### Section 2: 食物搭配
- 3 种主食和 2 种饮品的早餐搭配
- 发现搭配规律
- 学习乘法原理的应用

### Section 3: 数字排列
- 用 1、2、3 组成不同的两位数
- 学习树形图表示法
- 理解排列的有序性

### Section 4: 握手问题
- 3 人握手的组合问题
- 理解不重复计算的原则
- 掌握握手问题的计算方法

### Section 5: 练习与巩固
- 衣服搭配练习
- 数字组合练习
- 排队问题练习
- 路线选择练习

### Section 6: 总结与提升
- 本单元知识总结
- 课后挑战题

## 核心概念

### 搭配问题（乘法原理）
第一类有 m 种，第二类有 n 种，搭配总数 = m × n

### 排列问题
按照一定顺序排列，注意不重复、不遗漏

### 组合问题（握手问题）
n 个人握手，总次数 = (n-1) + (n-2) + ... + 1

## 解题方法
1. **连线法**：直观展示搭配关系
2. **列举法**：一一列出所有可能
3. **树形图**：清晰展示排列过程
4. **乘法计算**：快速得出搭配总数
