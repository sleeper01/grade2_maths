# 📐 二年级上册·第一单元「长度单位」动画课件

一个零依赖的 HTML 课件，用动画 + 图文结合的方式，帮助家长辅导孩子学习"长度单位"。

## 🚀 快速开始

直接双击打开 `index.html` 即可在浏览器中使用，**无需安装任何依赖**。

> 推荐使用 Chrome / Edge / Safari 最新版浏览器，分辨率 1280×800 及以上效果最佳。

## 📚 内容结构

| 章节 | 内容 | 时长建议 |
|------|------|---------|
| 🐻 第 1 节 | 认识厘米（1cm 的概念、用尺子量物体） | 10 分钟 |
| 🐻 第 2 节 | 认识米（1m 的概念、1m=100cm） | 10 分钟 |
| 🐻 第 3 节 | 认识线段（线段特征、画线段） | 8 分钟 |
| 🎮 第 4 节 | 选择单位（互动练习） | 5 分钟 |
| 🌟 单元小结 | 知识回顾 | 3 分钟 |

## 👨‍👩‍👧 家长使用建议

1. **一起看**：建议家长陪孩子一起观看，孩子小注意力容易分散
2. **慢慢讲**：每个动画都有"💡 家长读这句"的小提示，可以用温柔的语气读给孩子
3. **多暂停**：动画运行中可随时点击"↻ 重播"
4. **结合实物**：课件里出现的尺子、铅笔、线段都可以拿家里的实物对照看
5. **互动练习**：第 4 节让孩子自己选择答案，做对了记得表扬哦 🎉

## 🎯 学习目标

完成本课件后，孩子应该能够：
- ✅ 说出 1 厘米大约有多长
- ✅ 用尺子量较短的物体
- ✅ 知道 1 米 = 100 厘米
- ✅ 用米尺量较长的物体
- ✅ 识别和画出线段
- ✅ 根据物体长短选择合适的单位

## 🛠️ 技术说明

- 纯 HTML / CSS / JavaScript，零依赖
- 所有图形使用内联 SVG 绘制，无需外部图片
- 响应式设计，手机/平板/电脑均可使用
