# 二年级数学 第五单元：观察物体（一）

## 教学目标

1. 知道从不同位置观察同一物体，看到的形状可能不同
2. 能辨认从前面、侧面、上面观察到的简单物体的形状
3. 能正确数出由小正方体组成的立体图形的个数
4. 培养空间想象能力和观察能力

## 教学重点

### 1. 三个观察方向

| 方向 | 说明 | 示例 |
|------|------|------|
| 从前面看 | 正对着物体观察 | 看到物体的正面 |
| 从侧面看 | 从物体左侧或右侧观察 | 看到物体的侧面 |
| 从上面看 | 从物体上方俯视 | 看到物体的顶部 |

### 2. 重要结论

- **同一物体，从不同方向看，看到的形状可能不同**
- 球体是特例：从任何方向看都是圆形
- 正方体：从前面、侧面、上面看都是正方形

### 3. 观察组合体

数小正方体个数的方法：
1. 分层数（从下往上）
2. 分列数（从左往右）
3. 注意被遮挡的部分

## 生活实例

### 观察水杯

| 方向 | 看到的形状 |
|------|------------|
| 从前面看 | 长方形（带把手） |
| 从侧面看 | 长方形（把手在侧面） |
| 从上面看 | 圆形（杯口） |

### 观察汽车

| 方向 | 看到的形状 |
|------|------------|
| 从前面看 | 车头（前灯、车牌） |
| 从侧面看 | 车身（车门、车轮） |
| 从上面看 | 车顶（长方形） |

## 练习题

### 数一数

1. 2 层，每层 1 个 → **2 个**小正方体
2. 底层 2 个，上层 1 个 → **3 个**小正方体
3. 底层 4 个，上层 1 个 → **5 个**小正方体

### 判断对错

1. 从不同方向观察同一物体，看到的形状一定不同。（✗）
   - 解释：球体从任何方向看都一样

2. 观察球体时，从任何方向看都是圆形。（✓）

3. 从上面看正方体，看到的是正方形。（✓）

## 学习建议

### 培养空间想象能力

1. **实物观察**：
   - 观察家里的水杯、茶壶、玩具
   - 从不同方向看，画出看到的形状

2. **动手操作**：
   - 用积木搭建立体图形
   - 从前面、侧面、上面分别观察

3. **想象练习**：
   - 给出立体图形，想象从不同方向看到的形状
   - 给出视图，想象原物体可能的样子
