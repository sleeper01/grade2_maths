# 二年级数学上册课件

> 人教版小学数学二年级上册配套互动课件

## 📚 单元列表

| 单元 | 主题 | 内容说明 |
|------|------|----------|
| Unit 1 | 长度单位 | 认识厘米和米，学习测量物体长度 |
| Unit 2 | 100 以内的加减法 | 两位数的加减法运算 |
| Unit 3 | 角的初步认识 | 认识角、直角、锐角、钝角 |
| Unit 4 | 表内乘法（一） | 2-6 的乘法口诀 |
| Unit 5 | 表内乘法（二） | 7-9 的乘法口诀 |
| Unit 6 | 观察物体 | 从不同方向观察物体 |
| Unit 7 | 认识时间 | 认识钟表，学习整时和几时几分 |
| Unit 8 | 数学广角 | 搭配问题（排列组合初步） |

## 🚀 快速开始

### 方式一：本地打开
1. 克隆或下载本仓库
2. 在浏览器中打开任意单元的 `index.html` 文件
   ```
   grade2/maths/长度单位/index.html
   grade2/maths/angles-intro/index.html
   grade2/maths/time-telling/index.html
   ...
   ```

### 方式二：使用本地服务器
```bash
# 使用 Python
cd grade2/maths
python -m http.server 8080

# 使用 Node.js (需要先安装 http-server)
npx http-server -p 8080
```
然后在浏览器访问 `http://localhost:8080/`

## 📖 各单元简介

### 第 1 单元：长度单位
- 认识厘米（cm）和米（m）
- 学习使用尺子测量
- 理解 1 米 = 100 厘米
- 估测物体长度

### 第 3 单元：角的初步认识
- 认识角的各部分（顶点、边）
- 区分直角、锐角、钝角
- 学习用三角板判断角的类型
- 动手画角

### 第 4-5 单元：表内乘法
- 理解乘法的意义
- 学习 2-9 的乘法口诀
- 运用口诀进行计算
- 解决简单的乘法问题

### 第 6 单元：观察物体
- 从不同方向观察物体
- 辨认正面、侧面、上面
- 培养空间观念

### 第 7 单元：认识时间
- 认识钟面（12 个刻度、时针、分针）
- 学习认读整时
- 学习认读几时几分
- 理解 1 时 = 60 分

### 第 8 单元：数学广角 - 搭配问题
- 简单的排列组合
- 用连线法解决搭配问题
- 握手问题
- 数字排列问题
